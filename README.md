# pecat
This is the PE file analysis tool.

By the way, Kudo Haruka is very cute.

You can do 2 things.

1. Header analysis
2. Parse of string resource(.NET Application)

### Usage
- Header analysis

`gcc pecat.c -o pecat`

`pecat -c header FILE_NAME`

- Parse of string resource(.NET Application)

`gcc pecat.c -o pecat`

`pecat -c string FILE_NAME`
